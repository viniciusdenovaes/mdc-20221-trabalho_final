Códigos feitos pelo grupo Mining Minds para o trabalho final de mdc 2022/1

ATENÇÃO: O banco de imagens disponibilizado no google drive difere do banco de dados disponibilizado pelo site oficial
do artigo. Tanto no conjunto de treino, quanto no conjunto de teste. E também (ou somente) na quantidade de imagens.
Os experimentos feitos usando o banco de dados do google drive ficaram consistentemente com um resultado melhor (~0.03).
Enquanto que os feitos usando o banco de dados do artigo oficial ficaram pior (~0.04).
Os resultado apresentados no tabalho final foram feitos usando o banco de dados oficial, tendo um resultado de ~0.04

Pastas:

 * Data: arquivos relativos a dados (imagens), anotações dos dados e modelos salvos (com seus pesos).
 A maioria dos arquivos são ignorados pelo controle de versionamento (estarão ausentes na entrega do relatório)
 por causa do tamanho. Mas todos podem ser recuperados (imagens) ou gerados (modelos e pesos)

 * Prototipo: estudos feitos pelos integrantes do grupo, cada integrante do grupo tem uma pasta com seu nome
 e vários códigos em formato jupyter contendo estudo, visualização e geração de dados, prototipo de funções, etc...
 No momento da entrega do relatório será feito backup destes códigos individuais e teremos uma outra pasta com os
 protótipos limpos e organizados
 OBS: Todos os protótipos terão seu output apagado, por causa do tamanho das imagens geradas

 * src: código para produção de uma solução.
 Um programa que recebe uma imagem qualquer e escolha de um modelo de predição;
 e faz a extração de atributos da imagem
